普罗旺斯采摘季：用葡萄点缀秋色

返回列表　　作者：请叫我小花儿 　 2016-06-27

　　2016年6月22日，南非夏威夷果中国首场研讨会在上海威斯汀酒店成功举办，此次研讨会由南非Subtrop协会（即南非牛油果，夏威夷果，芒果和荔枝协会联盟）的市场发展经理Bonnie Buthelezi女士发起。南非夏威夷果协会会长Walter Giuricich先生及运营经理Barry Christie先生出席此次会议，并向到场来宾介绍南非夏威夷果种植生产、出口贸易及在中国市场的情况。

　　6月22日，南非夏威夷果中国首场研讨会在上海威斯汀酒店成功举办，此次研讨会由南非Subtrop协会（即南非牛油果，夏威夷果，芒果和荔枝协会联盟）的市场发展经理Bonnie Buthelezi女士发起。南非夏威夷果协会会长Walter Giuricich先生及运营经理Barry Christie先生出席此次会议，并向到场来宾介绍南非夏威夷果种植生产、出口贸易及在中国市场的情况。

　　来自Daxue Consulting公司的Matthieu David先生与大家分享了南非夏威夷果的中国市场调研。临安坚果炒货行业协会会长余中武主持了会议演讲嘉宾与观众互动环节，探讨了南非夏威夷果在中国市场的机遇与挑战。参会的企业就南非夏果的标准，采购，产量纷纷提问。南非协夏威夷果协会表示由于受干旱的影响，今年的产量减少了15%，明年也会继续受到影响，估计到2018年才能恢复正常。协会愿意制定更高的出口质量标准来满足中国市场的需求。会议共吸引了80余位坚果进口商，分销商，炒货商，和媒体。并得到了临安坚果炒货行业协会，南方坚果炒货行业协会以及云南坚果行业协会的大力支持。

　　夏威夷果含有丰富的钙、磷、铁、维生素B1、 B2和人体必需的8种氨基酸，有着“世界坚果之王”的美称。夏威夷果的英文名是Macadamia，由生物学家Ferdinand von Mueller为纪念自己的同事John Macadam而命名。可不要被“夏威夷果”这个名字给唬住了，其实它是“澳洲仁”。夏威夷果原产于澳大利亚，也叫昆士兰栗、澳洲胡桃，只是早期这种坚果在澳洲商业化种植程度不高，倒是被引进夏威夷后，大面积种植，成了当地特产，以其馥郁的香气和顺滑的口感席卷全球，澳洲坚果就以“夏威夷果”之名被世界所熟知。

　　不过夏威夷果的产地远不仅限于“老家”澳大利亚和“发迹地”夏威夷，在肯尼亚、南非、哥斯达黎加、危地马拉、巴西和中国很多地区都能见到夏威夷果树的身影。目前全球最大的夏威夷果产地，还当属南非——20世纪60年代，夏威夷果被引入南非以后，产业迅速发展。如今南非夏威夷果年出口量超过46000吨，产业规模已跃居世界首位。南非出产的夏威夷果品质也相当卓越——果肉饱满、颗粒圆润，口感细腻香滑不失酥脆，还带着浓郁香醇，充满热带森林的天然气息。

　　不仅美味满分，和很多益智养生的坚果一样，夏威夷果的健康功效也不容小觑。夏威夷果含油量高达60%-80%，并且都是胃肠道最容易吸收的油类，具有减少胃酸，阻止胃炎及十二指肠溃疡的功能。此外，夏威夷果富含的单不饱和脂肪酸能降低血压、调节和控制血糖水平、改善脂质代谢。夏威夷果对改善脑部营养也很有益处，特别适合孕妇和儿童食用。除了作为干果食用外，夏威夷果还是烘焙的好帮手，和巧克力搭配也是一绝，用来做食用油也丝毫不输。



南非夏威夷果种植者协会（SAMAC）

　　南非夏威夷果种植者协会由南非夏威夷果种植者，加工商及贸易商组成，其宗旨为“行业与种植者共同发展”。南非夏威夷果协会最初成立主要是为了应对病虫害问题，致力于对虫害、病害、用水和灌溉方式的各种研究项目的开展。随着产业的发展，如今南非夏威夷果协会已是一个具有多元化事业的组织。除了开展研究外，南非夏威夷果协会还为其成员提供种植技术服务，成为生产地区的主要技术顾问。营销和推广夏威夷果也是南非夏威夷果协会工作的重中之重。此外，南非夏威夷果协会还收集行业信息，进行数据统计，为市场决策提供有力的数据支持。
